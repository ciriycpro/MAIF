# MAIF — Multi-Agent Industrial Framework

**МАИФ — Мультиагентный индустриальный фреймворк.**

Открытый стандарт архитектуры индустриальных мультиагентных систем.

---

## Статус

- **Регистрация:** Роспатент, свидетельство о регистрации программы для ЭВМ № 2026611550 от 20 января 2026 года.
- **Правообладатель:** ООО «СИРИУС ПРО».
- **Лицензия:** Apache License 2.0.

## Назначение

Документ описывает требования к открытому стандарту индустриального мультиагентного слоя — рамки совместимости межагентного контура для систем разных вендоров, работающих с автономными агентами в индустриальной среде, включая объекты критической информационной инфраструктуры (КИИ).

## Совместимость и адаптация существующих систем

Стандарт не требует переписывания существующих индустриальных систем. Интеграция с легаси-системами (АСУ ТП, ОЗП, корпоративные шины) выполняется через адаптеры по паттерну миграции, аналогичному внедрению OPC UA.
