# Участие в развитии стандарта

## Статус документа

Стандарт МАИФ находится в фазе публикации спецификации и формирования рабочей группы.

## Формы участия

### Технические замечания и предложения

Вопросы по содержанию спецификации, выявленные противоречия или предложения по уточнению формулировок направляются через раздел Issues репозитория.

### Реализации стандарта

Стандарт открыт для независимых реализаций на любых языках программирования и в любых архитектурных стилях. Реализация считается совместимой при выполнении шести обязательных требований, определённых в спецификации (раздел [`/spec`](./spec)).

Авторы независимых реализаций могут уведомить правообладателя стандарта о созданной реализации для включения в реестр совместимых реализаций (раздел планируется к публикации).

### Участие в рабочей группе

Формирование рабочей группы по развитию стандарта осуществляется правообладателем по согласованию с профильными ведомствами и индустриальными центрами компетенций.

Контакт для участия в рабочей группе: через сайт правообладателя — https://ciriyc.ru
